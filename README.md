## 📝 Descripción

Proyecto desarrollado como práctica del Módulo 1 del Bootcamp de Desarrollo Web Fullstack en The Bridge.

Esta aplicación permite:

- Visualizar terremotos de la última semana en un mapa interactivo.
- Filtrar terremotos por su magnitud.
- Filtrar terremotos por un rango de fechas específico.
- Registro e inicio de sesión de usuarios mediante Firebase Authentication.
- Guardar y consultar terremotos favoritos en una base de datos de Firestore.
- Identificación visual de la intensidad mediante colores en los marcadores.

El objetivo es practicar conceptos fundamentales de **desarrollo front-end** e integración con servicios de terceros (APIs y Backend as a Service).

---

## 🛠 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla ES6)
- [Leaflet](https://leafletjs.com/) (Biblioteca para mapas interactivos)
- [Firebase](https://firebase.google.com/) (Autenticación y Base de Datos Firestore)
- [USGS Earthquake API](https://earthquake.usgs.gov/earthquakes/feed/v1.0/geojson.php) (Fuente de datos)
